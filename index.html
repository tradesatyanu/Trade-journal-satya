Option 1 — Python (no install needed):
cd ~/Downloads && python3 -m http.server 8000 Then open localhost:8000/trade_journal_v3.html

Option 2 — Node.js:
npx serve ~/Downloads Then open the localhost URL it shows.

Option 3 — VS Code:
Install the Live Server extension → right-click the file → Open with Live Server.
